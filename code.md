# Code

`code` - Visual Studio Code's command line utility.

## Help commands
```bash
code --help
```

## Open directory
```bash
code .      # open pwd
code <dir>  # general form
```

### Open file
```bash
code path/to/file
```

## Diff two files
```bash
code --diff /path/1 /path/2
# or
code -d /path/1 /path/2
```

## Show all extensions
```bash
code --list-extensions
```
