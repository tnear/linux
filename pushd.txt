PUSHD

Push directory onto directory stack.

# Push pwd onto stack:
[~] $ pushd
~

# Navigate away:
$ cd /etc

# Return to old pwd (~):
[/etc] $ popd
[~] $

# Push pwd onto stack and navigate elsewhere in one step:
[~] $ pushd /var/www
[/var/www] $ popd
[~] $

# All multiple directories to stack:
$ pushd /lib
$ pushd /var
$ pushd /etc

# -v = vertical view of stack:
$ dirs -v
0       /etc
1       /var
2       /lib
3       ~
