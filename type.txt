Type: Description of how an argument would run as a command.

# Describe 'ls':
$ type ls
ls is an alias for ls --color=auto

# Run non-aliased 'ls':
$ command ls

# Shell builtin example:
$ type jobs
jobs is a shell builtin

# 'mv' command:
$ type mv
mv is /usr/bin/mv
