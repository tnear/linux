paste - merge lines of files

$ head file*.txt
==> file1.txt <==
abc
def
ghi
==> file2.txt <==
123
456

# Combine lines using '-' as delimiter:
$ paste -d '-' file1.txt file2.txt
abc-123
def-456
ghi-

# -s = serial, paste one file at a time:
$ paste -d '-' -s file1.txt file2.txt
abc-def-ghi
123-456

# Join lines of file with comma:
$ paste -d ',' -s file1.txt
abc,def,ghi
