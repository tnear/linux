## Basic usage

```bash
# Describe 'ls'
$ type ls
ls is an alias for ls --color=auto

# 'mv' command:
$ type mv
mv is /usr/bin/mv

# Run non-aliased 'ls':
$ command ls
```

## Shell builtin example
```bash
$ type jobs
jobs is a shell builtin
```
