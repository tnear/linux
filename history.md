# history

`history` - GNU History Library

See also: [`!`](!.md), [`fzf`](fzf.md)

## View command history
```bash
$ history
```

### Search command history
```bash
$ history | grep grep
```

## Shortcuts
- `Ctrl+R` finds previous match (`bck-i-search`) 
- `Ctrl+S` finds next match     (`fwd-i-search`) (less widely supported)
