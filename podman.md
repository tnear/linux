# podman

`podman` - Simple management tool for pods, containers and images

Its CLI is almost identical to Docker.

See also: [`docker`](docker.md), [`docker-api`](docker-api.md)

## Podman machine
Podman machines are the lightweight Linux virtual machines that Podman uses to run Linux containers on macOS and Windows. They exist because containers depend on Linux kernel features, so Podman needs a Linux VM on those platforms. On Linux, podman machine is optional.
