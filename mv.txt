MV

mv - move (rename) files

# Rename file:
$ mv oldName newName

# Move file to directory (dir):
$ mv file.txt dir/

# Move multiple files to directory:
$ mv file1 file2 file3 dir/

# Rename directory:
$ mv oldDir newDir

# -i, --interactive = prompt to overwrite if dest file already exists:
$ mv -i a.txt b.txt
mv: overwrite 'b.txt'?
