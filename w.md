# w

`w` - Show who is logged on and what they are doing.

See also: [`who`](who.md)

## Basic usage
```bash
$ w
 11:12am up 608 day(s), 19:56,  6 users,  load average: 0.36, 0.36, 0.37
User     tty       login@  idle  what
smithj   pts/5      8:52am       w
jonesm   pts/23    20Apr06    28 -bash
```
